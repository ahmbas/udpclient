# udpclient
Simple Golang UDP Echo Server

```
NAME:
   echo-udp - UDP Echo Server

USAGE:
   udpclient [global options] command [command options] [arguments...]

VERSION:
   1.0.0

COMMANDS:
     help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --host value   Host to bind (default: "127.0.0.1")
   --port value   Port to listen to (default: 12345)
   --help, -h     show help
   --version, -v  print the version
```
